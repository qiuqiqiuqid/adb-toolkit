# 更新日志

## v1.0.1 (2026-02-28)

### 新增功能
- 批量安装 APK - 支持一次性选择多个 APK 文件安装
- 保留数据卸载 - 卸载应用时保留用户数据
- 批量卸载 - 勾选多个应用后一键批量删除
- Fastboot 解锁命令 - flashing unlock / oem unlock
- Fastboot 加锁命令 - flashing lock / oem lock
- 刷入分区 - 支持 boot / initboot / 自定义分区
- Fastboot 命令行 - 自定义执行任意 fastboot 命令
- 分辨率设置 - 自定义屏幕分辨率（宽 x 高）
- 显示密度设置 - 自定义 DPI
- 屏幕设置重置 - 一键恢复默认分辨率/密度

### 优化改进
- 设备信息增加更多内容：设备名(device)、CPU架构(abi)、存储空间、内存、序列号、Bootloader版本
- 精简代码结构
- 精简项目体积

### 界面调整
- "屏幕投屏"更名为"屏幕功能"
- Fastboot 页面重组：重启、解锁命令、刷入分区、命令行

## v1.0.0-test (2026-02-28)

### 初始版本
- 设备管理 - 检测设备、显示基本信息
- 应用管理 - 查看、安装、卸载、冻结第三方应用
- 文件管理 - 浏览设备文件、上传文件
- 屏幕投屏 - 基于 scrcpy，支持自定义帧率、码率、分辨率
- 工具箱 - 截图、Shell 命令、重启设备
- Fastboot - 设备检测、重启模式
