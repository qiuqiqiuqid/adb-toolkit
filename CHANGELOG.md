# 更新日志

## v1.0.3 (2026-02-28)

### 🔧 功能修复
- **界面点击响应修复** - 修复界面点击无响应问题
- **设备交互绑定** - 统一 Root、无线调试、重启等设备交互绑定
- **Root 功能** - 完善 Root 开关与状态显示
- **无线调试** - 新增无线调试 UI 控件，支持启用/连接/断开
- **文件管理** - 增强 Root 权限下的文件浏览功能
- **投屏功能** - 提升 scrcpy 启动的鲁棒性
- **重启功能** - 在设备信息区域新增重启按钮

### 🎨 UI 改进
- **事件绑定优化** - 使用 dataset.bound 标记避免重复绑定
- **错误处理增强** - 添加全局错误处理，防止初始化异常
- **状态提示完善** - 所有功能操作均有明确的 Toast 反馈
- **交互一致性** - 统一设备相关控件的点击响应

### 🚀 稳定性提升
- **设备信息展示** - 新增 Root 模式、无线调试、重启等控件
- **后端支持** - 完善 listFilesRoot 等相关 IPC 接口
- **容错性增强** - 关键操作增加异常处理与回退逻辑
